# brocli
brocli is a Chrome extension that turns Chrome's omnibar into a command line interface for navigating the web

## AC Commands

### List Command
* `-l (page)` - navigate to an entity's list page. Ex: `-l products`

### Edit Commands
* `-c (<id>)`- edit customer
* `-cat (<id>)` - edit category
* `-o (<id>)` - edit order
* `-p (<id>)` - edit product
